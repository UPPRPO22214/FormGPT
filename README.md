# Название проекта
## FormGPT
**Коротко**: «Конструктор опросов как Google Forms, но с встроенным GPT-помощником, который за минуты собирает структуру анкеты под вашу тему исследования и помогает не допускать ошибок формулировок».

[Смотреть видео](docs/cursorful-video-1765803387963.mp4)

## Функциональные требования (MVP)
1) Создание и редактирование опросов
Создать новый опрос: название, описание.
Типы вопросов: одиночный выбор, множественный выбор, текст , шкала (1–10).
Перетаскивание вопросов (drag & drop), копирование, удаление, редактирование.
Валидация обязательных вопросов.
Превью опроса “как респондент”.
2) GPT-помощник как стороннее API
Поле «Тема/задача исследования» + «Целевая аудитория».
Кнопка «Сгенерировать черновик анкеты» (Модель возравщает список вопросов с ответами)
Кнопка «Улучшить вопрос » для выделенного вопроса (Здесь будет задаваться промт для гпт).
Кнопка «Добавить проверочные вопросы» (Указываем количество)
3) Аналитика (базовая)
Дашборд по опросу: количество завершённых/незавершённых.
Экспорт в CSV/XLSX.
4) Управление и доступ
Регистрация/логин автора (email + пароль, OAuth).
Список опросов автора.

## Нефункциональные требования
Время ответа GPT ≤ 10 секунд при обычном запросе (без thinking)
Хранение данных: шифрование учетных записей в базе (хэши паролей с солью) .
UI: desktop
