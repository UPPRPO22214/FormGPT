# Название проекта
## FormGPT
**Коротко**: «Конструктор опросов как Google Forms, но с встроенным GPT-помощником, который за минуты собирает структуру анкеты под вашу тему исследования и помогает не допускать ошибок формулировок».

## Функциональные требования (MVP)
1) Создание и редактирование опросов
Создать новый опрос: название, описание, обложка (опц.).
Типы вопросов: одиночный выбор, множественный выбор, короткий текст, длинный текст, шкала (1–5/1–10).
Перетаскивание вопросов (drag & drop), копирование, удаление.
Валидация обязательных вопросов.
Превью опроса “как респондент”.
2) GPT-помощник (встроенный)
Поле «Тема/задача исследования» + «Целевая аудитория».
Кнопка «Сгенерировать черновик анкеты»: модель предлагает список блоков и вопросов с типами.
Кнопка «Улучшить формулировки» для выделенного вопроса (переписать без наводящих/двусмысленных формулировок).
Кнопка «Добавить проверочные вопросы» (контроль внимания/quality screeners).
3) Логика опроса (простая)
Переход к разделу/вопросу в зависимости от ответа (skip logic на 1 уровень).
4) Публикация и сбор ответов
Генерация публичной ссылки.
Базовая защита от спама (капча/одноразовый токен).
5) Аналитика (базовая)
Дашборд по опросу: количество завершённых/незавершённых.
Автоматические диаграммы для закрытых вопросов (распределения, %).
Экспорт в CSV/XLSX.
6) Управление и доступ
Регистрация/логин автора (email + пароль, OAuth).
Список опросов автора, статусы, дата обновления.
Дублирование опроса по шаблону.

## Нефункциональные требования
Время генерации черновика анкеты GPT ≤ 10 секунд при штатной теме.
Хранение данных: шифрование в базе.
UI: адаптивный (desktop/mobile).
